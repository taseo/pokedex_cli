# PokedexCLI

A REPL-style Pokedex application that uses the [PokéAPI](https://pokeapi.co) to fetch data about Pokémon

## Usage

- `go build && ./pokedexcli` will:
    - Compile the source code into an executable binary
    - Run the generated binary
- `go test ./...` will:
    - Run unit tests
